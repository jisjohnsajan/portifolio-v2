# Jis John Sajan — Portfolio

Personal portfolio website featuring a dark, modern design with smooth animations and interactive elements.

## Tech Stack

- **Next.js 16** — React framework with App Router
- **TypeScript** — Type-safe development
- **Tailwind CSS 4** — Utility-first styling
- **GSAP + ScrollTrigger** — Scroll-based animations
- **Lenis** — Smooth scrolling

## Features

- 🎯 Hero section with morphing text animation
- 🖱️ Custom cursor with hover states
- 📜 Scroll progress indicator
- 🎨 Glass-morphism card design
- 🔮 Scroll-triggered reveal animations
- 📜 Interactive education timeline
- 🖼️ Certificate lightbox with instant CSS toggle
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎬 Infinite marquee for skills
- 🔄 3D tilt cards
- 🧭 Auto-hiding navigation

## Sections

| Section | Description |
|---------|-------------|
| Hero | Animated name, morphing subtitle, stats |
| About | Bio, status badge, stat cards |
| Skills | 4 groups — Languages, Core, Tools & Tech, Soft Skills |
| Projects | 3 featured projects with images and GitHub links |
| Education | Timeline with 3 entries |
| Certifications | 9 certs with thumbnail grid + lightbox |
| Contact | Email CTA, resume download, social links |

## Getting Started

```bash
# Clone the repo
git clone https://github.com/jisjohnsajan/portifolio-v2.git

# Navigate to the project
cd portifolio-v2

# Install dependencies
npm install

# Run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
